Abstract

The Collatz map is deterministic, but its stopping-time landscape has irregular, heavy-tailed, and regime-dependent behavior. This project asks whether probabilistic machine learning can reveal stable statistical structure inside that deterministic complexity.

Research question

Can Collatz stopping times be modeled as structured random variables with arithmetic regimes, overdispersion, heavy tails, and calibrated predictive uncertainty?

Key idea

The key idea is to treat stopping times as count data and model them using Bayesian hierarchical methods, allowing residue classes and arithmetic regimes to affect both the mean structure and uncertainty.

Mathematical formulation

A simplified statistical model can be written as a Negative-Binomial generalized linear model for stopping times $\tau(n)$.

$$ \tau(n) \mid r(n), x(n) \sim \operatorname{NegBin}\big(\mu_n,\phi\big), \qquad \log \mu_n = \alpha_{r(n)} + \beta^\top x(n). $$

Here $r(n)$ denotes an arithmetic regime, such as a residue class, and $x(n)$ collects numerical or arithmetic covariates.

Main results

The project develops a Bayesian modeling pipeline for Collatz stopping times, including hierarchical Negative-Binomial GLMs, arithmetic regime conditioning, Monte Carlo calibration, posterior checks, and distributional diagnostics.
