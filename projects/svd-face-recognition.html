Abstract

This project explores the application of Singular Value Decomposition and Principal Component Analysis to two related problems: dimensionality reduction for structured tabular data and face recognition in high-dimensional image spaces. The first part uses PCA to reduce the dimensionality of a crime-rate dataset containing observations from 50 U.S. states and seven crime variables. The reduced representation is then used to construct a multivariate linear regression model for predicting homicide rates. The second part develops a facial-recognition algorithm based on the eigenfaces method: each new image is projected into a low-dimensional face subspace, and recognition is performed by comparing the projected coordinates with those stored in the database.

Keywords. Face Recognition, PCA, SVD, Linear Regression, Eigenfaces, Low-Rank Approximation.

Research question

How can spectral decompositions reduce the dimensionality of high-dimensional crime-rate and face-image data while preserving enough structure for prediction and recognition?

Key idea

The key idea is to replace raw high-dimensional observations with coordinates in low-rank spectral subspaces. For crime data, the relevant coordinates are principal components. For face images, the relevant coordinates are projections onto the face subspace generated by the dominant singular vectors of the centered image matrix.

Mathematical formulation

PCA constructs artificial variables as linear combinations of the original variables:

\[ \begin{array}{c} Y_1 = a_{10}+a_{11}X_1+\dots+a_{1p}X_p,\\ \vdots\\ Y_k = a_{k0}+a_{k1}X_1+\dots+a_{kp}X_p,\\ \vdots\\ Y_p = a_{p0}+a_{p1}X_1+\dots+a_{pp}X_p. \end{array} \]

The principal components are mutually uncorrelated, ordered by decreasing variance, and preserve total variance:

\[ \operatorname{Cov}(Y_i,Y_k)=0\quad(i\neq k), \qquad V(Y_1)\geq V(Y_2)\geq\dots\geq V(Y_p), \] \[ \sum_{i=1}^{p}V(X_i)=\sum_{i=1}^{p}V(Y_i). \]

The standardized observations are

\[ Z_{ij} = \frac{X_{ij}-\tilde X_j}{\sqrt{\operatorname{Var}(X_j)}}, \qquad \tilde X_j=\frac{1}{n}\sum_{i=1}^{n}X_{ij}. \]

The correlation matrix is

\[ R=\frac{Z^TZ}{n-1}. \]

Since the data are centered, each principal component has the form

\[ Y_i=\underline a_i^TZ. \]

The first component maximizes $V(Y_1)$ under $\underline a_1^T\underline a_1=1$. Since

\[ V(Y_1)=\underline a_1^T R\underline a_1, \]

the Lagrangian is

\[ \phi(\underline a_1,\lambda_1) = \underline a_1^TR\underline a_1 - \lambda_1(\underline a_1^T\underline a_1-1). \]

Differentiating gives

\[ (R-\lambda_1I_p)\underline a_1=0. \]

Therefore, the principal components are obtained from the eigenvalues and eigenvectors of $R$:

\[ V(Y_1)=\lambda_1\geq V(Y_2)=\lambda_2\geq\dots\geq V(Y_p)=\lambda_p. \]

The singular value decomposition of a data matrix $A$ is

\[ A=U\Sigma V^T. \]

Hence

\[ A^TA=V\Sigma^T\Sigma V^T, \qquad AA^T=U\Sigma\Sigma^TU^T. \]

For $R=\frac{Z^TZ}{n-1}$, the right singular vectors give the principal directions and the eigenvalues satisfy

\[ \lambda_i=\frac{\sigma_i^2}{n-1}. \]

The principal components are obtained as

\[ ZV=U\Sigma. \]

Crime data PCA

The crime-rate dataset is a matrix in $\mathbb{R}^{50\times 7}$, with 50 U.S. states and seven variables measured as crimes per 100,000 inhabitants: homicide, kidnapping, robbery, assault, burglary, theft, and motor vehicle theft.

The symmetric QR method gives the eigenvalues

\[ 4.1150,\quad 1.2387,\quad 0.7258,\quad 0.3164,\quad 0.2580,\quad 0.2220,\quad 0.1241. \]

Since only the first two eigenvalues are greater than one, the first two principal components are retained:

\[ Y_1 = 0.303Z_1+0.4318Z_2+0.3969Z_3+0.3967Z_4+0.4402Z_5+0.3575Z_6+0.2952Z_7, \] \[ Y_2 = 0.6292Z_1+0.1694Z_2-0.0423Z_3+0.3435Z_4-0.2022Z_5-0.4023Z_6-0.5024Z_7. \]

The first component acts as an overall danger index. The second component separates violent-crime structure from property-crime structure.

The rank-2 approximation is

\[ R_2 = U_{R,(:,1:2)} \Sigma_{R,(1:2,1:2)} V^T_{R,(:,1:2)}. \]

By the Eckart–Young theorem, this is the best rank-2 approximation. In the project,

\[ \|R-R_2\|=0.7258. \]

Regression model

The full multivariate linear regression model is

\[ y=X\beta+\epsilon, \qquad E(\epsilon)=0, \qquad \Sigma_\epsilon=\sigma^2I_n. \]

Least squares minimizes

\[ \phi(\beta)=(y-X\beta)^T(y-X\beta), \]

leading to the normal equations

\[ X^TX\beta=X^Ty. \]

If $X^TX$ is invertible, then

\[ \beta_{OLS}=(X^TX)^{-1}X^Ty. \]

For the complete regression model, the QR method gives

\[ \hat\beta_{OLS} = (0,\ 0.4445,\ 0.2690,\ 0.2806,\ 0.2511,\ -0.4958,\ -0.2424)^T, \]

with

\[ R^2_{adjusted}=0.5544. \]

The reduced regression model trained on the first two principal components has

\[ \beta_{PC_{OLS}}=(0,\ 0.3003,\ 0.6292), \]

and reaches

\[ R^2_{adjusted}=0.8555. \]

Thus, the principal-component model explains a larger portion of variance using two artificial variables instead of the seven original predictors.

SVD-based face recognition

Each face has $m\times n=M$ pixels and is represented as a column vector $f_i\in\mathbb{R}^M$. The training set of $N$ images is

\[ S=[f_1,\ldots,f_N]. \]

The mean face is

\[ \overline f=\frac{1}{N}\sum_{i=1}^{N}f_i. \]

The centered matrix is

\[ A=[f_1-\overline f,\ldots,f_N-\overline f]. \]

If $\operatorname{rank}(A)=r\leq N\ll M$, then

\[ A=U\Sigma V^T. \]

The columns $u_1,\ldots,u_r$ form an orthonormal basis for the face subspace. These vectors are the eigenfaces.

A new face $f$ is represented by

\[ x=[u_1,\ldots,u_r]^T(f-\overline f). \]

For each training face $f_i$:

\[ x_i=[u_1,\ldots,u_r]^T(f_i-\overline f). \]

Recognition is performed by minimizing

\[ \epsilon_i=\|x-x_i\|_2. \]

Instead of computing eigenvectors of $AA^T$, which is $M\times M$, the algorithm computes eigenvectors of the smaller matrix $A^TA$, which is $N\times N$:

\[ A^TA v_i=\mu_i v_i. \]

Since $A^TA$ and $AA^T$ share the same nonzero eigenvalues, the left singular vectors can be recovered from $AV$ and normalized.

The Olivetti dataset contains 10 images for each of 40 individuals, for $N=400$ images. Each image is grayscale with size $112\times92$ pixels. With a 0.7/0.3 train/test split, the full-rank eigenfaces classifier reaches

\[ 98.3333\%. \]

The accuracy $\theta$ depends nonlinearly on training-set size $s$ and rank $r$:

\[ \theta=f(s,r). \]

A possible exponential form is

\[ \theta(s,r)=1-\theta_0e^{-g(s,r)}. \]

The project proposes the explicit form

\[ \theta(s,r) = 1-k\cdot e^{-\left(\frac{\alpha s}{\beta+r}\right)}. \]

Results

The project selects two principal components, improves the adjusted coefficient of determination from $0.5544$ to $0.8555$, and obtains $98.3333\%$ full-rank eigenfaces accuracy on the Olivetti dataset.

Conclusions

Singular Value Decomposition is a powerful tool for high-dimensional data. In the crime-rate application, PCA reduces seven correlated variables to two interpretable directions and improves regression performance. In the face-recognition application, SVD builds a low-dimensional face subspace in which recognition is performed efficiently through nearest-neighbor comparison.

Future improvements include image preprocessing, scaling and alignment, comparison with convolutional neural networks, live video recognition, and more advanced spectral or quantum-inspired recognition methods.
